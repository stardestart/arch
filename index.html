Установка Arch Linux (0т новичка, таким же новичкам).

Предисловие

У меня не было опыта работы с linux системами, но в связи с постоянным ростом количества мне не нужных "стандартных" программ, появилось желание поменять windows на что-нибудь другое. Попробовав массу различных дистрибутивов linux, я решил учиться Linux с помощью Arch, он понравился мне максимальным минимализмом и схожестью с детским коструктором. Методом проб и ошибок я определил необходимый мне минимум пакетов для повседневной работы.
Хочу поделиться своим опытом с такими же новичками в linux как и я сам, на понятном для новичков языке.
Важно знать, что после этой установки все данные с дисков будут удалены.

Подготовка к установке.

  1. Заходим на wiki.archlinux если там изложено понятно, то лучше использовать официальную wiki.
  2. Определяем какой режим загрузки использует ваш ПК, UEFI или Legacy Boot.
  3. Скачиваем образ.
  4. Скачиваем rufus.
  5. Записываем образ на флешку, при перезагрузке загружаемся с флешки. (Если есть uefi, то выбираем загрузку через uefi).
  6. Для установки необходимо постоянное интернет соединение, это одна из особенностей Arch, он устанавливает обновленные пакеты.
    Если ПК подключён к сети через Wi-Fi.
    Подключаем устройство wlan0 к wifi сети ИмяСети.
    #: iwctl station wlan0 connect ИмяСети
    Вводим пароль wifi сети.
    #: ПарольСети
    Проверка соединения, 3 раза прозвон ya.ru.
    #: ping -c 3 ya.ru
  7. Выбираем русскую раскладку.
    #: loadkeys ru
  8. Установка шрифта с кириллицей.
    #: setfont cyr-sun16
  9. Синхронизация системных часов.
    #: timedatectl set-ntp true
